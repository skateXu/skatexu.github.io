    python数据类型、编码类型的转化

    misc
    发布时间 : 2023-10-12 20:45
    字数:412 阅读 :
    1. str和bytes
    2. int和bytes
    3. 进制和ascii

    str和bytes

    >>> str_ = "PK\x03\x04"
>>> str_.encode()
b'PK\x03\x04'

>>> bytes_ = b'PK\x03\x04'
>>> bytes_.decode()
'PK\x03\x04'
    >>> str_ = "504B0304"
>>> bytes.fromhex(str_)
b'PK\x03\x04'

>>> bytes_ = b'PK\x03\x04'
>>> bytes_.hex()
"504b0304"

    举例：以hex形式，读取文件和写入文件

    with open('1.txt', 'rb') as f:
    str_ = f.read().hex()

# str_ = "504B0304"
str_1 = ""
for i in range(0, len(str_), 2):
    str_1 += str_[i:i+2]

with open('2.txt', 'wb') as f:
    f.write(bytes.fromhex(str_1))

    int和bytes

    >>> bytes_ = b"flag{}"
>>> int.from_bytes(bytes_, 'big')
112615676672893

>>> int_ = 112615676672893
>>> int_.to_bytes(10, 'big')
b'\x00\x00\x00\x00flag{}'
    >>> from Crypto.Util.number import *

>>> long_to_bytes(int_)

>>> list(bytes_)

>>> list_ = [102, 108, 97, 103, 123, 125]
