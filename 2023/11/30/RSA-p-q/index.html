    在近几个月的比赛里，经常遇见给出p^q的题型，在此总结一下。

    核心思路就是对p、q从高位开始逐位进行剪枝爆破，并加上以下约束条件：

    1. \(leak = p \oplus q\)，p和q每个二进制位之间的关系都可以根据异或值来调整。
    2. 当定下了p和q的高位，如果低位全部补0，\(pq < n\)；全部补1，\(pq > n\)。

    核心函数pq_xor如下（适用于下面所有题型）：

    def pq_xor(p="", q=""):
    lp, lq = len(p), len(q)
    tp0 = int(p + (512-lp) * "0", 2)
    tq0 = int(q + (512-lq) * "0", 2)
    tp1 = int(p + (512-lp) * "1", 2)
    tq1 = int(q + (512-lq) * "1", 2)

    if tp0 * tq0 > n or tp1 * tq1 < n:
        return
    if lp == leak_bits:
        pq.append(tp0)
        return

    if xor[lp] == "1":
        pq_xor(p + "0", q + "1")
        pq_xor(p + "1", q + "0")
    else:
        pq_xor(p + "0", q + "0")
        pq_xor(p + "1", q + "1")

    \(leak = p \oplus q\)

    题目源码

    from Crypto.Util.number import *

p = getPrime(512)
q = getPrime(512)
n = p * q
leak = p ^ q

print(f"n = {n}")
print(f"leak = {leak}")

    分析

    p、q直接异或，因为p和q的高1位肯定是1，所以异或后的leak肯定没有512位，为了方便使用脚本，可以用zfill在前面补0。

    另外，因为p、q可以互换位置，所以它们在不同分支上分别被解出，最终得到的pq列表包含p和q。

    解题

    leak_bits = 512
xor = bin(leak)[2:].zfill(512)
pq = []

pq_xor()
print(pq)

    \(leak = p \oplus (q >> 16)\)

    题目源码

    from Crypto.Util.number import *

p = getPrime(512)
q = getPrime(512)
n = p * q
leak = p ^ (q >> 16)

print(f"n = {n}")
print(f"leak = {leak}")

    分析

    p、q错位异或，这样处理后leak和p的高16位是相同的，直接传入leak的高16位作为初始的p。而且leak肯定是512位，这里就不需要补0了。

    另外，跟上面不同，因为这种情况的p和q不可以互换位置，所以得到的pq列表只包含p。

    解题

    leak_bits = 512
xor = bin(leak)[2:]
pq = []

pq_xor(xor[:16])
print(pq)

    \(leak = (p \oplus q) >> 100\)

    题目源码

    from Crypto.Util.number import *

p = getPrime(512)
q = getPrime(512)
n = p * q
leak = (p ^ q) >> 100

print(f"n = {n}")
print(f"leak = {leak}")

    分析

    这个leak不全，少了低100位，这样解出来的p、q也会少了相应的位数，剩余位数用coppersmith恢复即可。

    解题

    leak = leak << 100
leak_bits = 412
xor = bin(leak)[2:].zfill(512)
pq = []

pq_xor()
print(pq)

for p_high in pq:
    x = PolynomialRing(Zmod(n), 'x').gen()
    f = p_high + x
    res = f.monic().small_roots(X=2**100, beta=0.4)
    if res:
        p = int(f(res[0]))
        print(p)
        break

    \(leak = (p \oplus q)\ \%\ 2^{502}\)

    题目源码

    from Crypto.Util.number import *

p = getPrime(512)
q = getPrime(512)
n = p * q
leak = (p ^ q) % 2**502

print(f"n = {n}")
print(f"leak = {leak}")

    分析

    因为爆破过程是从高位开始，这个leak缺了高10位（其实也不一定是10位，不过没关系），所以需要爆破这些位。

    解题

    leak_bits = 512

for i in range(2**10):  # 爆破 leak 的高位
    xor = bin(leak + (i << 502))[2:].zfill(512)
    pq = []

    pq_xor()
    if pq:
        print(pq)
        break

    \(leak = (p \oplus (q >> 8))\ \%\ 2^{502}\)

    题目源码

    from Crypto.Util.number import *

p = getPrime(512)
q = getPrime(512)
n = p * q
leak = (p ^ (q >> 8)) % 2**502

print(f"n = {n}")
print(f"leak = {leak}")

    分析

    在这种情况下，当q右移的位数小于leak缺的高位的位数时，解出来的pq列表可能有多个错误值，所以需要遍历一下并且加上if n % p == 0来判断。

    解题

    leak_bits = 512

for i in range(2**9, 2**10):  # 爆破 leak 的高位
    xor = bin(leak + (i << 502))[2:].zfill(512)
    pq = []

    pq_xor(xor[:8])
    if pq:
        for p in pq:
            if n % p == 0:
                print(p)
                exit()
